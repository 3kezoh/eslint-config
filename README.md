# @ekezoh/eslint-config

A collection of ESLint rules that I use in my projects.

## Installation

```bash
pnpm add --save-dev eslint @ekezoh/eslint-config-layout
```

## Usage

Add the following to your `.eslintrc`:

```json
{
  "extends": ["@ekezoh/eslint-config-layout"]
}
```
